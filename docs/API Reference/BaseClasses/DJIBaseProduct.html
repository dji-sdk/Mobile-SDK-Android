      Package:dji.sdk.base
      Doc Key:DJIBaseProduct
      Description:

      Abstract class for all DJI Products. Aircraft and HandHeld objects are sublcasses of BaseProduct and can be accessed from getProduct in DJISDKManager. Additional components can be found in Aircraft and HandHeld that are unique to those products only.
