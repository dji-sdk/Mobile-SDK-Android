      Doc Key:DJIWaypointMission_DJIWaypointMissionOperator_DJIWaypointMissionEvent
      Description:

      List of all possible events that occur during the life cycle of a waypoint mission

      Enum Members:
      UNKNOWNUnknown event.
      INITIALIZEDThe waypoint abstraction is initialized and recovered from the disconnection.
      NAVIGATION_MODEThe aircraft enters navigation mode.
      EXIT_NAVIGATION_MODEThe aircraft exits navigation mode. If there is executing mission, the mission will be intterupted (state changes from executing to NotReady).
      RC_MODE_CHANGEDUser changes the Remote Controller switch. If there is executing mission, the mission will be intterupted (state changes from executing to NotReady if multi-mode is enabled or state changes from executing to ReadyToUpload if multi-mode is disabled).
      SIMULATION_OFFSimulation mode is changed from on to off.
      DISCONNECTEDThe connection is broken.
      UPLOAD_FAILEDError happens during the upload.
      UPLOAD_PROGRESS_UPDATEThe upload progress is updated.
      UPLOAD_DONEThe mission is uploaded to the aircraft completely.
      DOWNLOAD_FAILEDError happens during the download.
      DOWNLOAD_PROGRESS_UPDATEThe download progress is updated.
      DOWNLOAD_DONEThe mission is uploaded to the aircraft completely.
      EXECUTION_STARTEDThe mission execution has started.
      EXECUTION_START_FAILEDStarting of the execution failed.
      EXECUTION_PAUSEDThe mission was paused
      EXECUTION_PAUSE_FAILED
      EXECUTION_STOPPEDThe mission was stopped
      EXECUTION_STOP_FAILEDStopping the mission failed
      EXECUTION_RESUMEDA previously paused execution was resumed
      EXECUTION_RESUME_FAILEDResuming failed.
      EXECUTION_PROGRESS_UPDATEThe execution progressed and generated an update.
      EXECUTION_INTERRUPTEDThe mission execution was interrupted.
      EXECUTION_FINISHEDThe mission execution finished. Normal completion.

      enum WaypointMissionState
      Package:dji.sdk.missioncontrol
      Supported By:A3+LB2, Inspire 1, Inspire 1 Pro/Raw, Inspire 2, Matrice 100, Matrice 600, Matrice 600 Professional, Mavic Pro, N3+LB2, Phantom 3 4K, Phantom 3 Advanced, Phantom 3 Professional, Phantom 3 Standard, Phantom 4, Phantom 4 Professional.
      Not Supported By:Osmo, Osmo +, Osmo Mobile.
      SDK Key:None
      Doc Key:DJIWaypointMission_DJIWaypointMissionOperator_DJIWaypointMissionState
      Description:

      List of all states a mission may go through during its life cycle.

      Enum Members:
      UNKNOWNUnknown State.
      NOT_READYThe current state is not ready for uploading a mission.
      READY_TO_UPLOADIt is OK to start uploading a waypoint mission.
      UPLOADINGThe uploading is started successfully. Detail information for each waypoint is being uploaded one by one.
      READY_TO_EXECUTEWaypoint mission is uploaded completely and the aircraft is ready to start the execution.
      EXECUTINGThe execution is started successfully.
      EXECUTION_PAUSEDWaypoint mission is paused successfully. User can call resume to continue the execution.
      EXECUTION_STOPPINGThe stop command has been called and we didn't receive the ACK from the aircraft yet. It is NOT allowed to call another command in this state. The next state is either disconnected or the previous state (executing or paused).
