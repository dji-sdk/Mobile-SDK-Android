Class Overview

The class represents a follow-me mission. User can use it to make the aircraft follow the GPS device, like remote controller with GPS signal or mobile device.

public static void updateFollowMeCoordinate (double targetLatitude, double targetLongitude, float altitude, DJIBaseComponent.DJICompletionCallback callback)

Updates the coordinate the aircraft will follow. Once the follow me mission is initialized, this method is used to continuously update the coordinate to follow. If the aircraft doesn't receive an update within 6 seconds, it will hover in position until the next update arrives. This is the only property or method in this class that can communicate with the aircraft during a mission. All other properties and methods are used offline to prepare the mission which is then uploaded to the aircraft.

Parameters
targetLatitude The latitude position the aircraft will follow. Should be within 200m of current location.
targetLongitude The longitude position the aircraft will follow. Should be within 200m of current location.
altitude The altitude position the aircraft will follow.
callback Return the execution result.
