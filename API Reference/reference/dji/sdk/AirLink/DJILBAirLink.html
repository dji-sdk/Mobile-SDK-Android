public abstract void setPIPDisplay (DJILBAirLink.LBAirLinkPIPDisplay outputMode, DJIBaseComponent.DJICompletionCallback callback)

Sets the secondary video output Picture in Picture (PIP) display mode. The Lightbridge module can connect to both an FPV camera (through the HDMI and AV ports) and a camera mounted on the HD Gimbal (through the Gimbal port). The output video can then be a combination of the two video sources. Either a single video source can be displayed, or one can be displayed within the other as a Picture in Picture (PIP). If the mode is set incorrectly, then no output video will be displayed. For example, if only a FPV camera is connected, or the FPV bandwidth is set to 100 percent, then the only mode that will display data is LBAirLinkPIPDisplayLB.

Parameters
outputMode Picture in Picture display mode.
callback The execution callback with the execution result returned.
